How it works
  1. Upload a .zip file downloaded from the DHS Program website.
  2. The system parses the filename to identify the country, year, and survey programme.
  3. Stata .DTA files are extracted and loaded into typed PostgreSQL tables.
  4. Variable names, labels, and value mappings are preserved in the metadata catalog.
  5. Already-loaded files are automatically skipped (safe to re-upload).
